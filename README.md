<h1 align="center"> Smart Parking Dashboard</h1>

<p align="center">
  IoT-Based Real-Time Parking Management System with Intelligent Vehicle Detection
</p>

<hr/>

<h2> Overview</h2>
<p>
Traditional parking systems lack real-time visibility, leading to traffic congestion,
fuel wastage, and inefficient space utilization in urban areas.
This project introduces an intelligent IoT-enabled solution that dynamically monitors
parking availability and provides real-time insights through a centralized web platform,
enhancing efficiency and supporting smart city infrastructure.
</p>

<hr/>

<h2> Dashboard Features</h2>
<ul>
  <li> Real-time parking slot monitoring</li>
  <li> Available & Occupied slot visualization</li>
  <li> Live vehicle detection integration</li>
  <li> Occupancy analytics and statistics</li>
  <li> Instant updates using IoT data</li>
</ul>

<hr/>

<h2> Tech Stack</h2>
<ul>
  <li><b>Frontend:</b> React.js + Tailwind CSS</li>
  <li><b>Backend:</b> Node.js (optional)</li>
  <li><b>Database:</b> Firebase / MongoDB</li>
  <li><b>IoT:</b> Arduino UNO R4 WiFi</li>
  <li><b>Detection:</b> OpenCV / YOLO</li>
</ul>

<hr/>

<h2> How It Works</h2>
<ol>
  <li> Sensors detect parking slot occupancy</li>
  <li> Data is transmitted to the server</li>
  <li> System processes and updates status</li>
  <li> Dashboard reflects real-time changes</li>
</ol>
